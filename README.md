# Nandita Perumal's Website

This repository contains the source code for Nandita Perumal's personal website and portfolio, hosted on GitHub Pages.

## Project Overview

The website showcases Nandita's work, projects, and personal updates. It is built using HTML5, CSS3 (Sass), and JavaScript.

## Project Structure

- **`index.html`**: The main landing page of the website.
- **`anniversary-2026.html`**: A dedicated page for the 2026 anniversary.
- **`generic.html`**: A template for generic content pages.
- **`elements.html`**: A reference page for various HTML elements and styles used across the site.
- **`assets/`**: Contains static assets including:
  - `css/`: Compiled CSS stylesheets.
  - `js/`: JavaScript files.
  - `sass/`: SCSS source files for styling.
  - `webfonts/`: Font files.
- **`images/`**: Directory for project images and media.

## Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/nanditaperumal/nanditaperumal.github.io.git
   ```

2. Open `index.html` in your browser to view the site.

3. To modify styles, edit the files in `assets/sass/` and recompile to `assets/css/`.

## Deployment

This site is deployed via GitHub Pages. Any changes pushed to the `main` branch will be automatically reflected on the live site.
